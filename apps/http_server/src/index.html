Features & Goals

  • Microkernel architecture (OS services are isolated processes)
  • Userspace-first design: make OS development approachable for everyone
  • Run performance-critical services in kernel (safely)
  • Schema-less and async message passing IPC
  • Fast edit-and-run cycle for hassle-free developer experience
